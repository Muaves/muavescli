commands
Everything you need,
one flag away.

Simple, memorable flags. No man page required.

muaves
Show the help message and available commands
muaves -oA
Display the "About Me" section with bio and info
muaves -oP
List all projects with descriptions and links
muaves -oH
Show help — same as running with no args
muaves -OW
Open muaves.com in your default browser
muaves -H:l
Show links and pick one to open interactively
muaves --version
Print the current installed version
muaves --v
Shorthand for --version
installation
Up and running
in seconds.

Requires Python 3.6+. No external dependencies.

Windows
1
Download and extract the ZIP file
2
Open Command Prompt or PowerShell
3
Navigate to the windows folder and run muaves.bat
4
To use from anywhere, run install.bat
# After install muaves -oA
Linux / macOS
1
Download and extract the ZIP file
2
Open a terminal and navigate to the linux folder
3
Run ./muaves to launch
4
To use from anywhere, run ./install.sh
# After install muaves -oP
